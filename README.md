# API Stand Tests

Proyecto de pruebas automatizadas para la API de Urban Grocers.

## 📌 Descripción
Este proyecto contiene pruebas automatizadas para validar la creación de usuarios en la API, verificando diferentes escenarios positivos y negativos.

## 🛠️ Tecnologías utilizadas
- Python
- Pytest
- Requests

## 📂 Estructura del proyecto
- `configuration.py` → Configuración de URLs del servicio
- `data.py` → Datos de prueba
- `sender_stand_request.py` → Funciones para enviar solicitudes HTTP
- `create_user_test.py` → Pruebas automatizadas
